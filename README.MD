# MasterMind - MiniProjeto de C - 1ºPI - EPMS

## Como descarregar o programa?

 - Ir à opção Code e depois fazer o download do zip ou fazer o clone do projeto com ``git clone``;

 - Após abrir o código, aceder ao terminal e escrever:
 ```text
 gcc main.c -o MasterMind
 ```

- Executar o ``.exe``;

- Jogar e divertir :)
